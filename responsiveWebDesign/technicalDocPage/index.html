O que é Python ?

Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi lançada por Guido van Rossum em 1991. Atualmente possui um modelo de desenvolvimento comunitário, aberto e gerenciado pela organização sem fins lucrativos Python Software Foundation. Foi considerada pelo público a 3ª linguagem "mais amada", de acordo com uma pesquisa conduzida pelo site Stack Overflow em 2018, e está entre as 5 linguagens mais populares, de acordo com uma pesquisa conduzida pela RedMonk. Devido às suas características, ela é principalmente utilizada para processamento de textos, dados científicos e criação de CGIs para páginas dinâmicas para a web.

Instalando Python

Provavelmente, se você é usuário de MacOS ou Linux você pode pular essa parte do tutorial pois Python já vem instalado nesses sistemas operacionais. Se você é usuário Windows e ainda não instalou Python, siga os passos abaixo para instala-lo.

  1. Primeiro vamos baixar o Python. Para isso, clique aqui para acessar a pagina oficial do Python. Nessa pagina, passe o mouse por cima da opção "Downloads" e clique em "Windows"

  2. Agora clique no primeiro link dessa pagina referente ao download da versão mais recente do Python. No meu caso, a versão mais recente é a 3.9.1, porém, para você essa versão pode ser mais recente do que a minha.

  3. Depois de baixar o instalador, execute-o e prossiga a instalação. Marque a opção de adicionar o Python para o PATH para evitar problemas futuros. Selecione a opção de instalação padrão.

  4. Para verificar se o Python foi instalado corretamente, abra o terminal e escreva "python". Se ele foi instalado corretamente, o seu terminal responderá de forma parecida com a imagem abaixo. Se alguma mensagem de erro dizer que o comando "python não foi reconhecido", tente instala-lo novamente. Caso você estaja usando MacOS ou Linux, teste esse comando no terminal para verificar se realmente o Python está instalado.

Editor de texto

Antes de começarmos a criar nossos códigos em Python, é necessário que você tenha algum editor de texto/código. Se ainda não tiver nenhum editor, recomendo que instale o Visual Studio Code, pois as imagens que utilizarei nos próximos tópicos serão desse editor de código, e para se familiarizar-se melhor e interessante que você utilize esse mesmo editor. Caso você tenha algum problema ao instalar ou baixar o editor, tente instala-lo assistindo esse vídeo do canal "Mão no código"

Olá, mundo !

Depois de instalar o Python e um editor de texto, vamos criar nosso primeiro código, que será o clássico "Olá, mundo !".

  1. Crie uma arquivo no seu editor de código com o final igual a ".py". O arquivo pode ter qualquer nome, mas a extensão dele precisa ser "py". Exemplo: "primeiro_codigo.py" ou "ola_mundo.py"

  2. Com esse arquivo aberto no editor de texto escreva o comando 

    print("Olá, mundo !")
    O comando print() é conhecido como uma função, e nesse caso, ela é responsável por "imprimir" textos na tela do seu computador. Essa função aceita como parâmetro um texto qualquer, e nesse caso o texto é "Olá, mundo !", mas ele poderia ser qualquer outro texto (não esquecer de colocar as aspas, se esquecer de colocar ocorrerá problemas.

  3. Agora o código ja está pronto, só falta executarmos ele. Para isso, se você estiver usando o editor Visual Studio Code, clique em "Run" no menu do canto superior, e depois em "Run without debuging". Ao realizar isso, seu programa será executado e o resultado será mostrado no canto inferior do editor de texto.

Variáveis

Agora vamos falar um pouco sobre variaveis, que são muito importantes para qualquer linguagem de programação. Logo abaixo esta uma lista de variaveis e como declara-las.

  • int: variável que armazena números inteiros (ex: 100, -100, 1346123, -129424, etc).
    Para criar uma variável do tipo "int" basta escrever o código abaixo (o nome da variável é importante para poder utiliza-la depois, e pode ser qualquer texto que não seja uma palavra reservada pela lunguagem e nem um texto com inicio numérico). OBS: No código abaixo foi utilizado o número 10, porém, poderia ser qualquer outro número inteiro

    nome_variavel = 10

  • float: variável que armazena números flutuantes (com casas decimais) OBS: casas decimais são representadas com "." ao invés de ",". Exemplo de float: 1.23, -23.2, 12.0, 12456.1256

    nome_variavel = 12.5

  • string: variável que armazena textos (pode ser representadas com ' ou "). Ex: "Olá, mundo !", 'Tudo bem ?', "Como vai ?"

    nome_variavel = 'Olá, mundo !'

  • boolean: armazená valores booleanos, que no caso pode ser 2 valores: True (verdadeiro) ou False (falso).

    nome_variavel = True

Existem outras variáveis, porém essas são as mais básicas e essenciais para o começo.
